# Nebula

Nebula 是一款 macOS 数据库与消息系统工作台，面向 MySQL、TiDB、PostgreSQL、MongoDB、Elasticsearch、Kafka、MQTT、Redis、Etcd、TDengine 等连接的日常数据浏览、结构查看、控制台执行、导出与 AI 辅助操作。

## 下载

请在 Releases 中下载最新的 `Nebula-*.dmg` 安装包。

## 在线更新

Nebula 内置在线更新检查。安装后可在菜单中点击：

```text
nebula > 检查更新...
```

如果有新版本，应用会展示更新说明并引导下载、安装和重启。

这个仓库仅作为 Nebula 安装包与在线更新发布仓库使用，不包含应用源代码。
